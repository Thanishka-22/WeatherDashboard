
🌦 Weather History Web Application  

📌 Project Description  
This project is a responsive Weather History Web Application built using ReactJS and Axios.  
It allows users to retrieve historical weather data for any city by selecting a specific date, month, or year.  
The application fetches data from the Open-Meteo Archive API and displays temperature, humidity, pressure, and weather conditions in a clean and user-friendly interface.  

Features
Search weather data by city  
Get weather data by:  
Specific Date  
Full Month  
Full Year  

🌡 Displays:
Maximum Temperature  
Minimum Temperature  
Average Humidity  
Mean Sea Level Pressure  

Weather Code  
Dynamic data rendering  
Responsive UI Design  
Fast API integration using Axios  

🛠 Technologies Used  
ReactJS  
JavaScript (ES6)  
Axios  
CSS  
Open-Meteo Archive API  

📈 Future Improvements  
Add weather charts (Graph visualization)  
Add loading spinner  
Add error handling alerts  
Add export to PDF option  
Deploy using Vercel / Netlify
